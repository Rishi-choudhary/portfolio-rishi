# Rishi Choudhary — Personal Portfolio

A single-page personal portfolio website for Rishi Choudhary — a Computer Science student, Shopify developer, and AI/ML builder. 

Inspired by minimalist, editorial design principles (Ref: kirillz.com), this site focuses on typography, generous whitespace, and a high-signal content layout.

## 🚀 Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Typography:** Playfair Display (Display), DM Sans (Body), JetBrains Mono (Accents)
- **Animations:** Custom IntersectionObserver-based reveal animations
- **Theming:** Dark mode support with persistence
- **Deployment:** [Vercel](https://vercel.com/)

## 🛠️ Key Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop.
- **Selective Work Gallery:** Showcasing major projects like the Smart Parking System (SIH Winner) and VibeQA.
- **Experience Timeline:** Detailed work history including internships and freelance projects.
- **Recognition Section:** Highlighting awards and certifications.
- **Dark Mode:** System-aware theme toggle with a minimalist UI.

## 🚦 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
